# IP Intelligence Lookup

A small web app that takes an IP address and opens multiple threat-intel sites in new tabs with that IP.

Enter an IPv4 or IPv6 address and click **Search all sources** to open each site in a new tab with that IP or open each site individually.

**Note:** To open all tabs at once, allow popups or redirects for this site in your browser (e.g. use the address bar icon or site settings). Otherwise the browser may block the new tabs.
